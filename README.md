### 🚀 Sobre mim

Sou estagiário de TI atualmente trabalhando com **automação predial** e **infraestrutura de recarga de veículos elétricos**, unindo desenvolvimento de software a hardware real em campo. Meu foco principal está no **frontend** e **web design**, mas gosto de mergulhar no fullstack quando necessário.

- 🔧 Trabalho com projetos fullstack (Python + React)
- ⚡ Desenvolvi um **CSMS** baseado em **OCPP 1.6** para carregadores EV
- 🇧🇷 Brasileiro | Idiomas: Português (nativo) e Inglês (técnico)

---

### 🛠️ Projeto em destaque: **March System Controller**

Uma plataforma full‑stack de automação predial (Modbus TCP) com módulo de carregamento de VEs integrado.

| Camada | Stack |
|--------|-------|
| **Backend** | Python + FastAPI |
| **Frontend Desktop** | React + TypeScript |
| **Mobile** | React Native |
| **Carregadores EV** | OCPP 1.6 |
| **Protocolo predial** | Modbus TCP |

**Funcionalidades recentes:**
- ✅ Whitelist de autorização por ID tag
- ✅ Correção de bug crítico de loop infinito
- ✅ Perfis de carregamento inteligente (`SetChargingProfile`, `MeterValues`, `StatusNotification`)
