### 🚀 Sobre mim

Sou estagiário de TI atualmente trabalhando com **automação predial** e **infraestrutura de recarga de veículos elétricos**, unindo desenvolvimento de software a hardware real em campo. Porém meu foco de estudo gira em torno de aplicações frontend, com foco em web design.

- 🔧 Trabalho atualmente com projetos fullstack, desenvolvendo aplicações em Python e React
- ⚡ Desenvolvi um **CSMS (Charging Station Management System)** baseado em **OCPP 1.6**, integrado a carregadores EV
- 🇧🇷 Brasileiro, trabalho principalmente em português e inglês

---

### 🛠️ Projeto em destaque: **March System Controller**

Uma plataforma full-stack de automação predial baseada em Modbus TCP, com o módulo de carregamento de VEs integrado como serviço.

| Camada | Stack |
|---|---|
| **Backend** | Python + FastAPI |
| **Frontend Desktop** | React + TypeScript  |
| **Mobile** | React Native  |
| **Carregadores EV** | OCPP 1.6 |
| **Protocolo predial** | Modbus TCP |

**Funcionalidades recentes:**
- ✅ Whitelist de autorização por ID tag para os carregadores
- ✅ Correção de bug crítico de loop infinito no fluxo de criação de andares
- ✅ Perfis de carregamento inteligente (Smart Charging) com `SetChargingProfile`, `MeterValues` e `StatusNotification`
